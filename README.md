# Automated Invoice System

A cloud-based invoicing tool that automates the creation and distribution of invoices.

## Features
- **Invoice Management:** View, search, and filter client invoices.
- **PDF Generation:** Generate professional PDF invoices (mocked in this version).
- **Status Tracking:** Monitor paid, pending, and overdue invoices.
- **Revenue Insights:** Basic financial forecasting dashboard.

## Setup

### Backend (Node.js)
1. Go to `backend/`
2. Run `npm install`
3. Run `npm start` (Runs on port 5003)

### Frontend (Next.js)
1. Go to `frontend/`
2. Run `npm install`
3. Run `npm run dev`

## Tech Stack
- **Backend:** Node.js, Express
- **Frontend:** Next.js, Tailwind CSS, Lucide React
